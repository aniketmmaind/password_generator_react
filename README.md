# Password Generator (React)

A small, responsive password generator built with React and Vite. Use it to quickly create secure passwords with configurable length and character sets, copy results to the clipboard, and evaluate password strength.

> Lightweight single-page app scaffolded with Vite and React.

## Features

- Generate secure random passwords
- Configure password length
- Toggle inclusion of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Copy password to clipboard with one click
- Simple strength indicator
- Responsive UI (works well on desktop and mobile)

## Tech stack

- React (with Vite)
- Tailwind CSS 
- Plain JavaScript / JSX
